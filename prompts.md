# Engenharia de Prompt

Prompt Engineering consiste em criar instruções claras para obter melhores respostas de um LLM.

## Boas Práticas

* Ser específico
* Informar contexto
* Definir objetivo
* Utilizar exemplos

## Exemplo

Ruim:

"Fale sobre Linux."

Melhor:

"Explique os conceitos básicos de Linux para um profissional de suporte técnico iniciando seus estudos."

## Conceitos Aprendidos

* Contexto
* Instruções
* Exemplos (Few-shot)
* System Prompt
