# O que é um LLM?

LLM significa Large Language Model.

São modelos treinados com grandes quantidades de texto capazes de compreender, interpretar e gerar linguagem natural.

## Exemplos

* ChatGPT (OpenAI)
* Llama (Meta)
* Mistral
* Gemma (Google)
* DeepSeek

## Capacidades

* Responder perguntas
* Gerar textos
* Traduzir idiomas
* Resumir documentos
* Auxiliar programação

## Conceitos Aprendidos

* Tokens
* Context Window
* Prompt
* Inferência
* Modelos Open Source
