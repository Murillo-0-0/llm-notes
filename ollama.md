# Ollama

Ollama é uma ferramenta que permite executar Large Language Models localmente.

## Vantagens

* Open Source
* Privacidade dos dados
* Não depende de APIs externas
* Controle total do ambiente

## Modelos Populares

* Llama 3
* Mistral
* Gemma
* DeepSeek
* Phi

## Exemplos

Executar um modelo:

```bash
ollama run llama3
```

Listar modelos:

```bash
ollama list
```

## Conceitos Aprendidos

* Inferência local
* Modelos Open Source
* Hospedagem de LLMs
* IA local
