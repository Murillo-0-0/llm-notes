# O que é RAG?

RAG significa Retrieval-Augmented Generation.

É uma técnica que permite que um LLM consulte informações externas antes de gerar uma resposta.

## Fluxo Simplificado

Usuário → Busca de Informações → LLM → Resposta

## Vantagens

* Menos alucinações
* Informações atualizadas
* Conhecimento específico da empresa
* Melhor precisão das respostas

## Exemplo

Um chatbot corporativo pode consultar documentos internos antes de responder ao usuário.

## Conceitos Aprendidos

* Base de conhecimento
* Busca semântica
* Vetorização
* Contexto externo
