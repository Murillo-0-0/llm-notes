# O que é um Agente de IA?

Um agente utiliza um LLM para tomar decisões e executar ações.

Diferente de um chatbot tradicional, um agente pode interagir com ferramentas externas.

## Exemplos

* Consultar banco de dados
* Executar scripts
* Chamar APIs
* Ler documentos
* Automatizar tarefas

## Fluxo Simplificado

Usuário → Agente → Ferramenta → Resultado

## Conceitos Aprendidos

* Planejamento
* Execução de tarefas
* Ferramentas (Tools)
* Automação
